Automated deployment of a fully scalable WordPress application on AWS using Terraform. Includes VPC networking, security groups, RDS database, Application Load Balancer, and EC2 instances with auto-scaling configuration.

- Infrastructure organized as modules (main.tf, provider.tf, variables.tf, outputs.tf)
- Custom userdata.sh for automated server configuration
- Complete networking setup with public/private subnets
